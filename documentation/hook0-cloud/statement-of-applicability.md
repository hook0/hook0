## Scope description

Information security regarding development, sales and hosting of Hook0 software solutions.  
We process and protect the following data:

- Customer data
- Financial and HR data

The following processes are (partially) outsourced, but fall within the scope of the ISMS:

- Hosting
- Software development (DCVS)
