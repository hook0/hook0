Issue #6: Org Dashboard 12px Text Too Small

Labels and metadata text at 0.75rem (12px) are hard to read

Before (screenshot)

Problem: Two CSS classes use font-size: 0.75rem (12px) and one uses 0.6875rem (11px):
  • .org-dashboard__label -- 0.75rem (12px) -- "Organization", "Currently limited to", "Developer plan notice"
  • .org-dashboard__quota-label -- 0.6875rem (11px) -- "members", "applications", "events per day", "days retention"
At standard screen DPI, 12px is below comfortable reading size. 11px is genuinely hard to read.
