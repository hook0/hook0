Issue #10: "Set up a new team workspace" Text Too Small

Description text under "New Organization" card is smaller than similar text elsewhere

Before (screenshot)

Problem: The "New Organization" dashed card uses .org-selector__apps-count for its description text, which is styled at 0.6875rem (11px) and --color-text-tertiary. Compare with the empty state description "Create your first application to start sending events." which renders at 0.8125rem (13px) via Hook0EmptyState. The text "Set up a new team workspace" should match that size for consistency.

Proposals

Current (11px)
+
New Organization Set up a new team workspace
Current: .org-selector__apps-count at 0.6875rem (11px). Hard to read, inconsistent with the empty-state description in sibling cards.
A -- Match empty state (13px)
+
New Organization Set up a new team workspace
Add a dedicated class for the new-org description at 0.8125rem (13px) with --color-text-secondary instead of tertiary. Matches the Hook0EmptyState description size. Minimal change: just stop reusing .org-selector__apps-count for this element.
B -- Slightly larger (14px), lighter weight
+
New Organization Set up a new team workspace
Use 0.875rem (14px) with font-weight: 400. Same size as the title but distinguished by lighter weight and secondary color. Makes the card feel more like a first-class CTA rather than an afterthought.

Implementation

The "New Organization" card reuses .org-selector__apps-count (line 374-376 of OrganizationAndApplicationSelector.vue) for the description. The fix is to either: (a) add a new class .org-selector__new-org-description with the correct size, or (b) use the existing Hook0EmptyState description styling approach. Option (a) is a 3-line CSS addition.
