Issue #7: "Limited to" Cards -- Too Much Vertical Space

Quota cards waste space when they wrap to multiple rows

Before (screenshot)

Problem: The "Currently limited to" section uses Hook0Stack layout="grid" grid-size="compact" gap="sm", rendering 4 nested Hook0Card components, each with its own Hook0CardContent padding. The cards have border + padding + inner icon-badge + stacked text, creating a lot of vertical bulk. When wrapping to 2x2 on narrower screens, the total height is excessive for what amounts to 4 small numbers.

Keep the card structure but reduce Hook0CardContent padding from default to a tighter value. Change gap="sm" to a custom smaller gap (~0.5rem). Reduce Hook0IconBadge from size="md" to size="sm". Saves ~30% vertical height.
B -- Replace cards with flat stat pills (recommended)

U 5 members

D 30 days retention
Remove the nested Hook0Card wrappers entirely. Replace with a simple flex-wrap container of inline pill elements. Each pill is icon + number + label on one line. No borders, just a subtle background. Dramatically reduces vertical space -- wrapping is now a single row height per line instead of full card height.

Most compact option. No cards, no background -- just an inline horizontal list with pipe separators. Each stat is value + label on a single baseline. Minimal vertical footprint. Wraps naturally. Removes all nested card overhead.
